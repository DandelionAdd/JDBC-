

---

## 适用场景

- 大学数据库课程设计
- MySQL + Java 入门练习
- JDBC 基础操作参考

---

## 技术栈

- **后端语言**：Java
- **数据库**：MySQL
- **构建工具**：Maven
- **JDBC**：用于 Java 与 MySQL 的交互

---

## 模块说明

| 模块     | 作用                         |
|----------|------------------------------|
| `dao`    | 封装对数据库的增删改查操作   |
| `model`  | 定义与数据库表对应的实体类   |
| `service`| 实现业务逻辑，调用 dao       |
| `ui`     | 提供用户交互界面             |
| `util`   | 提供数据库连接等通用工具     |

---

## 使用方式

1. 导入 MySQL 数据库（使用 `sql/` 目录下的脚本）
2. 修改 `util` 中的数据库连接配置（URL、用户名、密码）
3. 使用 Maven 构建项目
4. 运行 `Main.java` 启动程序
